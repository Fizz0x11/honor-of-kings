## 🎮 项目介绍

**Warcraft Coach** 是一个基于 OpenCode 平台的王者荣耀 AI 教练技能，汇聚职业级游戏理解，助你从青铜迈向王者！

### ✨ 核心特性

- 📚 **峡谷经济学** - 深入解析经济来源与分均经济标准
- 🗺️ **半途打法** - 职业级全局节奏体系，上下半途精确判断
- 📈 **五路发育公式** - 中路、对抗路、发育路、打野、辅助各位置指南
- ⏰ **时间轴节奏** - 30秒到20分钟的完整战术指南
- 👁️ **视野博弈** - 草丛机制与三种视野布置技巧
- ⚔️ **团战技巧** - 夹位、拆火与射手生存法则

# 🏆 Warcraft Coach - Honor of Kings Professional Coach

> *"Understand the game, carry your team."*

A comprehensive AI coaching skill for Honor of Kings (王者荣耀), built for the OpenCode platform. Learn professional-level game knowledge including economy management, rotation timing, lane strategies, and team fight positioning.

## ✨ Features

- 📚 **Valley Economics** - Deep dive into gold sources and income standards
- 🗺️ **Half-Map Strategy** - Pro-level macro play, precise top/bottom map calls
- 📈 **5-Lane Farming Formulas** - Position-specific guides for all roles
- ⏰ **Timeline Rhythm** - Complete tactics from 30s to 20min
- 👁️ **Vision Control** - Bush mechanics and vision setup
- ⚔️ **Team Fight Techniques** - Positioning, focus fire, ADC survival

## 📖 Usage

This skill activates when you ask about Honor of Kings strategies:

- "How to play ADC?"
- "Lane matchup tips?"
- "What is half-map strategy?"
- "Team fight positioning?"
- Any question about Honor of Kings...
