# 射手对线解析

> 来源：多个攻略综合整理

## 鲁班 vs 公孙离

### 判断
- 鲁班腿短无位移，公孙离位移多爆发高
- **鲁班永远无线权 = 永远是劣势方**

### 核心口诀
```
无线权 = 反方向跑 + 探草走
有线权 = 蹭伞打她 + 抢资源
```

### 具体行动表

| 场景 | 鲁班怎么做 |
|------|------------|
| 公孙离压你 | **往塔下缩**，等她伞落地那1-2秒打她一下 |
| 她推线过来 | **塔下补刀**，不追出去 |
| 打野来gank | **往反方向跑**，不要贪 |
| 中路miss了 | **往塔下走**，可能有人在蹲你 |
| 抢完线了 | **往中路靠**，跟队友一起走 |

### 一句话总结
**活到六神装 = 赢** 🐢

## 虞姬 vs 公孙离

### 判断
- 虞姬2技能物理免疫，克制公孙离平a输出
- **虞姬天然有线权 = 优势方**

### 核心口诀
```
她a你 = 你开2硬抗
她放伞 = 你开2冲上去干！
```

### 具体行动表

| 公孙离动作 | 虞姬应对 |
|------------|----------|
| 公孙离a你 | **开2技能硬抗**，她打不掉你的血 |
| 公孙离放伞落地 | **立刻开2技能冲上去打她**（这是她最弱的时候） |
| 公孙离1技能（法术伤害）| **走位躲**，2技能挡不了这个 |
| 公孙离想逃跑 | **开大追击**，黏住她打 |

### 有线权三件事（虞姬全做）

| 优先级 | 虞姬行动 |
|--------|----------|
| 1️⃣ 有额外资源 | **全力靠过去**，开2技能冲 |
| 2️⃣ 无额外资源 | **赶紧回中抢下一波线** |
| 3️⃣ 半靠边路 | **注意公孙离1技能**，走位躲法术伤害 |

### 一句话总结
**虞姬打公孙离 = 她a你你开2，她放伞你冲上去干！** 🔥

## 射手通用生存法则

- **前期弱如鸡**：叫辅助连体婴保发育
- **中期强如虎**：趁对面没魔抗赶紧打团
- **后期稳如狗**：边缘OB等时机，别冲进人群

### 发育路节奏
- 跟团推进后**立刻回中路顶线**
- 再收野区（避免被蹲草）
- 口诀：**「推进→顶中→收野→再抱团」**
